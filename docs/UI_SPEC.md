# UI/UX Спецификация — Momentum

---

## 1. Общая структура приложения

```
┌─────────────────────────────────────────────────────┐
│  NAVBAR: [⚡Momentum]  [Идеи]  [Планы]  [Расписание] │
├─────────────────────────────────────────────────────┤
│                                                     │
│              CONTENT AREA                           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- Навигационная панель фиксирована сверху (Bootstrap `navbar navbar-dark bg-dark`)
- Логотип слева: иконка + "Momentum"
- Три пункта меню справа: Идеи, Планы, Расписание
- Активный пункт: `nav-link active`
- Фон страницы: Bootstrap `bg-dark` (`#212529`), текст `text-light`

## 2. Страница "Идеи" (/ideas)

```
┌─────────────────────────────────────────────┐
│ Идеи                    [+ Новая идея]       │
├─────────────────────────────────────────────┤
│  [+ Новая идея]  ← кнопка перед первой      │
├─────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────┐ │
│ │ ⠿  Создать свою компанию          [2]   │ │
│ │    Хочу основать технологический стар.. │ │
│ │    Создано: 1 апр 2025    [✎] [🗑]       │ │
│ └─────────────────────────────────────────┘ │
│  [+ Новая идея]  ← кнопка после карточки   │
│ ┌─────────────────────────────────────────┐ │
│ │ ⠿  Написать книгу                 [0]   │ │
│ │    ...                                  │ │
│ └─────────────────────────────────────────┘ │
│  [+ Новая идея]                             │
└─────────────────────────────────────────────┘
```

### Карточка идеи
- Bootstrap card: `card bg-dark border-secondary mb-2`
- Drag handle (⠿) слева — `cursor: grab`
- Заголовок: `h5 card-title text-white` — кликабельный (ведёт на /ideas/:id)
- Описание: `card-text text-secondary` — обрезается на 150 символах
- Бейдж планов: `badge bg-secondary` — "2 плана"
- Дата создания: `small text-muted`
- Кнопки действий: иконки `btn btn-sm btn-outline-secondary`

### Кнопки "Новая идея"
- `btn btn-sm btn-outline-primary` с иконкой "+"
- Появляются между карточками и до/после списка
- Стиль: слегка прозрачный, становится ярче при hover

### Состояние drag-and-drop
- Перетаскиваемая карточка: `opacity: 0.5, box-shadow`
- Placeholder (место вставки): пунктирная рамка `border-dashed border-primary`

## 6. Детальная страница плана (/plans/:id)

```
┌─────────────────────────────────────────────┐
│ ← Все планы                                 │
│ ← Идея: Создать свою компанию  (если есть)  │
│                                             │
│ Исследование рынка               [✎] [🗑]   │
│ ─────────────────────────────────────────── │
│ Описание плана...                           │
│                                             │
│ Задачи (3)                                  │
│  [+ Новая задача]  ← вверху                 │
│ ┌─────────────────────────────────────────┐ │
│ │ ⠿ #1  Анализ конкурентов               │ │
│ │       Изучить 5 основных конкурентов   │ │
│ │  [🔵 Новая] [📅2] [💬1]       [✎] [🗑]  │ │
│ └─────────────────────────────────────────┘ │
│  [+ Новая задача]                           │
│ ┌─────────────────────────────────────────┐ │
│ │ ⠿ #2  Опрос потенциальных клиентов     │ │
│ │       ...                               │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### Карточка задачи
- Порядковый номер: `#1`, `#2` — автоматически по позиции
- Статус: цветной `badge`:
  - `new` → `bg-secondary` серый
  - `in_progress` → `bg-warning text-dark` жёлтый
  - `done` → `bg-success` зелёный
  - `done_partially` → `bg-info text-dark` голубой
  - `abandoned` → `bg-danger` красный
- Клик на карточку → `/plans/:planId/tasks/:taskId`

## 8. Страница "Расписание" (/schedule)

```
┌─────────┬──────────────────────────────────────┐
│ ЛЕВАЯ   │  ПРАВАЯ ПАНЕЛЬ                       │
│ ПАНЕЛЬ  │                                      │
│         │  Воскресенье, 6 апреля 2025 [Сегодня]│
│ [◄] Апрель 2025 [►]                            │
│         │  Задачи этого дня:                   │
│ МАРТ    │  ┌────────────────────────────────┐  │
│ ─────── │  │ 10:00–12:00                    │  │
│ Пн...Вс │  │ Анализ конкурентов             │  │
│ ...     │  │ [Исследование рынка]           │  │
│  3  2   │  │ Статус: 🔵 Новая       [✕]     │  │
│         │  └────────────────────────────────┘  │
│ АПРЕЛЬ  │  [+ Добавить задачу на этот день]     │
│ ─────── │                                      │
│ Пн...Вс │  Лента (с 7 апреля):                 │
│ ...     │  ─── 7 апреля ───                    │
│ [6] 3   │  14:00–16:00 Опрос клиентов          │
│         │  ─── 8 апреля ───                    │
│ МАЙ     │  09:00–11:00 Анализ конкурентов      │
│ ─────── │                                      │
│ ...     │                                      │
└─────────┴──────────────────────────────────────┘
```

### 8.1 Левая панель

**Навигатор месяца:**
- `← [Апрель 2025] →` — кнопки `btn btn-sm btn-outline-secondary`
- Текущий выбранный месяц выделен

**Три сетки месяцев:**
- Каждая сетка: заголовок месяца + таблица `table table-dark table-sm`
- Шапка: Пн Вт Ср Чт Пт Сб Вс (в Bootstrap: `text-muted small`)
- Ячейка дня:
  - Обычная: цифра + бейдж задач (если > 0)
  - Текущая дата: `bg-primary rounded-circle text-white`
  - Выбранная: `border border-primary rounded`
  - Прошедшая: `text-muted`
  - Hover: `bg-secondary cursor-pointer`
- Пустые ячейки (дни другого месяца): пустые

**Бейдж задач в ячейке:**
- `badge bg-info text-dark` — маленький, в правом нижнем углу ячейки

### 8.2 Правая панель

**Заголовок дня:**
- Полное название дня: "Воскресенье, 6 апреля 2025"
- Если сегодня: `badge bg-success` — "Сегодня"
- Если прошедший день: `badge bg-secondary` — "Прошедший день" + иконка замка 🔒

**Карточка слота (будущая дата):**
```
┌─────────────────────────────────────┐
│ 10:00 – 12:00                  [✕] │
│ Анализ конкурентов                  │
│ 📋 Исследование рынка               │
│ 🔵 Новая                      [✎]  │
└─────────────────────────────────────┘
```

**Карточка слота (прошедшая дата):**
```
┌─────────────────────────────────────┐
│ 10:00 – 12:00                  🔒  │
│ Анализ конкурентов                  │
│ 📋 Исследование рынка               │
│ Комментарий:                        │
│ ┌─────────────────────────────────┐ │
│ │ Немного поработал...            │ │
│ └─────────────────────────────────┘ │
│                    [Сохранить]      │
└─────────────────────────────────────┘
```

**Форма добавления задачи (раскрывается по кнопке):**
```
┌─────────────────────────────────────┐
│ Добавить задачу на 6 апреля         │
│                                     │
│ План: [Выберите план ▾]             │
│ Задача: [Выберите задачу ▾]         │
│ Начало: [10:00]  Конец: [12:00]     │
│                                     │
│              [Отмена] [Добавить]    │
└─────────────────────────────────────┘
```

## 9. Уведомления

**Toast (in-page):**
- Bootstrap Toast `toast` в правом нижнем углу
- Тёмный фон `bg-dark border-warning`
- Иконка ⏰
- Текст: "Задача через 5 минут: [название]"
- Не закрывается автоматически
- Кнопки: "Открыть расписание", "✕"

---

## 10. Общие компоненты

### Диалог подтверждения удаления
- Bootstrap modal: `modal-sm`
- Текст: "Вы уверены, что хотите удалить ...?"
- Кнопки: [Отмена] [Удалить] (btn-danger)

### Spinner загрузки
- Bootstrap spinner `spinner-border text-primary`
- Центрирован в контейнере

### Пустое состояние списка
- Иконка + текст "Пока нет идей. Создайте первую!"
- `text-muted text-center py-5`

### Badges статусов (переиспользуемый компонент)
- Принимает статус, возвращает Badge с нужным цветом и текстом

---

## 11. Адаптивность

- Bootstrap grid: `container-fluid`
- Расписание на мобильных: левая и правая панели — вертикальный стек
- Все карточки: 100% ширины на мобильных
- Таблицы календаря: `table-sm` + `font-size: 0.8rem` на мобильных
